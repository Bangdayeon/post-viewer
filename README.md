# 👀 post-viewer

상태 관리 도구 연습용 프로젝트

**JSONPlaceholder**를 사용하여 상태 관리 도구를 활용한 실습을 해봅니다

## 🗝 스택

- React
- Typescript

## 🛠 상태 관리 도구

- 🌴 [TanStack Query](https://ldd6cr-adness.tistory.com/303)
- axios
- 🐻 [Zustand](https://ldd6cr-adness.tistory.com/297)
- Github Action
